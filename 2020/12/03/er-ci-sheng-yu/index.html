给定n和奇素数p，求x满足x2≡n(mod p)x^2 \equiv n (mod \ p)x2≡n(mod p)

首先考虑有没有解，有解的n称为二次剩余，无解的n称为非二次剩余
根据费马小定理，当p为质数时，满足np≡n(mod p)n^p \equiv n (mod \ p)np≡n(mod p)
由于p是奇质数，则n2p−12≡1(mod p)n^{2\frac{p-1}{2}} \equiv 1 (mod \ p)n22p−1​≡1(mod p)
即np−12n^{\frac{p-1}{2}}n2p−1​为1在模p意义下开根的结果，只有1和-1两种可能
有欧拉准则，当np−12≡1n^{\frac{p-1}{2}} \equiv 1n2p−1​≡1时，n是二次剩余，否则是非二次剩余。
当n是二次剩余时，易得x2p−12≡xp−1≡1x^{2\frac{p-1}{2}} \equiv x^{p-1} \equiv
1x22p−1​≡xp−1≡1成立，必要性得到。
充分性以及非二次剩余的证明鸽了。

再考虑有解时有几个解，设两个不同的解x1,x2x_1,x_2x1​,x2​，则满足x12≡x22x_1^2 \equiv
x_2^2x12​≡x22​，即x12−x22≡(x1+x2)(x1−x2)≡0x_1^2-x_2^2 \equiv (x_1+x_2)(x_1-x_2)
\equiv 0x12​−x22​≡(x1​+x2​)(x1​−x2​)≡0，由于x1≠x2x_1 \neq
x_2x1​​=x2​，只有可能x1和x2互为相反数，由于p是奇数，x1和x2的奇偶性一定不同，因此值一定不同。所以除0外每个二次剩余都对应两个解，且这两个解互为相反数。同样的，两个相反数对应了一个二次剩余，因此在[1,p−1][1,
p-1][1,p−1]中总共有p−12\frac{p-1}{2}2p−1​个二次剩余

令i2≡a2−ni^2 \equiv a^2-ni2≡a2−n
有(a+i)p+1≡n(a+i)^{p+1} \equiv n(a+i)p+1≡n

  • ip≡i(a2−n)p−12≡−ii^p \equiv i(a^2-n)^{\frac{p-1}{2}} \equiv -iip≡i(a2−n)2p−1​≡−i

    (a+i)p+1≡(ap+ip)(a+i)≡(a−i)(a+i)≡n(a+i)^{p+1} \equiv (a^p+i^p)(a+i) \equiv
    (a-i)(a+i) \equiv n(a+i)p+1≡(ap+ip)(a+i)≡(a−i)(a+i)≡n

ll TS(ll n, ll p)
{
    if(n==0) return 0;
    if(p==2) return (n&1) ? 1 : -1;
    if(Pow(n, p>>1, p)!=1) return -1;
    if(p&2) return Pow(n, (p+1)>>2, p);
    int s = __builtin_ctzll(p^1);
    ll q = p>>s, z = 2;
    for(; Pow(z, p>>1, p)==1; z++);
    ll c = Pow(z, q, p);
    ll r = Pow(n, (q+1)>>1, p);
    ll t = Pow(n, q, p), tmp;
    for(int m=s, i; t!=1; )
    {
        for(i=0, tmp=t; tmp!=1; i++) tmp = tmp*tmp%p;
        for(; i<--m; ) c = c*c%p;
        r = r*c%p;
        c = c*c%p;
        t = t*c%p;
    }
    return r;
}
void solve()
{
    scanf("%d%d", &n, &p);
    ll ans = TS(n, p);
    if(ans==-1) puts("Hola!");
    else if(!ans) puts("0");
    else
    {
        ll ans2 = p - ans;
        if(ans>ans2) swap(ans, ans2);
        printf("%lld %lld\n", ans, ans2);
    }
}
