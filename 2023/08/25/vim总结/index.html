vim 总结

jyf111
2023-08-25 (Updated: 2023-08-28)
15 mins to read

¶Neovim配置

¶安装

Ubuntu apt安装的版本较旧，所以选择自行安装0.9.1版本。

¶卸载

¶配置

¶起点

选择NvChad作为配置的起点。

在custom/init.lua中设置vim.g.mapleader = ","，将<leader>设置为逗号。

插件配置参考ayamir/nvimdots、lazyvim和BrunoKrugel/dotfiles。

下面只介绍涉及到快捷键或者命令的插件，作为备忘。

¶插件管理

使用folke/lazy.nvim。

  • 打开Lazy面板：:Lazy

¶用户界面

¶主题

使用NvChad自带的base46。

  • 选择主题：<leader>th

选用catppuccin，并设置transparency=true（可选）。

¶面板

使用NvChad自带的NvDash，设置load_on_startup=true（可选）。

¶窗口浏览
  • 向下切分：:sp
  • 向右切分：:vsp
  • 上下左右移动：<C-h/j/k/l>
  • 关闭当前窗口：:q
¶Buffer line

使用NvChad自带的tabufline。

  • 下一个Buffer：<Tab>
  • 上一个Buffer：<S-Tab>
  • 关闭当前Buffer：<leader>x
¶平滑滚动
  • 向上滚动半页：<C-u>
  • 向下滚动半页：<C-d>
  • 向上滚动一页：<C-b>
  • 向下滚动一页：<C-f>
  • 将当前行置于顶部：zt
  • 将当前行置于中央：zz
  • 将当前行置于底部：zb

¶工具

¶备忘单
  • NvCheatSheet: <leader>ch
¶文件树

插件：nvim-tree/nvim-tree.lua

:h nvim-tree-mappings-default

  • :NvimTreeToggle: <C-n>
  • :NvimTreeFocus: <leader>e
  • 上下移动：j/k
  • 进入文件夹/打开文件：<CR>
  • 关闭文件夹：<BS>
  • 预览：<Tab>
  • 退出：q
  • 创建文件：a
  • 删除文件：d
  • 重命名文件：r
  • 标记：m
  • 复制文件：c
  • 粘贴文件：p
  • 剪切文件：x
  • 搜索：S
¶文件查找

插件：nvim-telescope/telescope.nvim

  • 查找文件：<leader>ff
  • 查找文件中的内容：<leader>fw
  • 查找打开的Buffer中的文件：<leader>fb
  • 查找所有文件：<leader>fa
  • 查找git commit：<leader>cm
  • 查找git status：<leader>gt
  • 查找帮助文件：<leader>fh
  • 查找主题：<leader>th
  • 上下移动：<Up>/<Down>或在Normal模式下j/k
  • 选择：<CR>
  • 在预览窗口上下移动：<C-u>/<C-d>
  • ?：打开快捷键面板
¶终端

使用NvChad自带的Nvterm。

  • 在下方打开/关闭一个终端：<A-h>
  • 在右方打开/关闭一个终端：<A-v>
  • 打开/关闭一个浮动终端：<A-i>
¶问题列表
  • 切换列表：TroubleToggle <leader>tt
¶行跳转
  • 跳转到某行：:<number>，该插件可以提供实时预览功能

¶编辑

¶注释

插件：numToStr/Comment.nvim

  • 切换当前行注释：gcc
  • 切换行注释：gc
  • 切换块注释：gb
¶高亮

插件：nvim-treesitter/nvim-treesitter

  • 查看所有语言安装情况：:TSInstallInfo
  • 更新：TSUpdateSync
  • 安装：TSInstall lang
  • 卸载：TSUninstall lang
  • 增量选择功能：
    • 扩大选择：<CR>
    • 缩小选择：<BS>
¶todo 高亮

插件：folke/todo-comments.nvim

  • TODO:
  • PERF:
  • NOTE:
  • FIX:
  • WARN:
  • TEST
  • 列出所有todos：:TodoTrouble <leader>td
  • 查找todos：:TodoTelescope <leader>ts
¶快速跳转

根据位置提示快速跳转。

  • 集成到了搜索 (/)中
  • 按s/S（s是全局，S是treesitter中的所有父节点），然后输入任意前缀

¶语言

¶LSP

插件：neovim/nvim-lspconfig（配置）、williamboman/mason.nvim（安装管理）、williamboman/mason-lspconfig.nvim

  • 查看LSP状态：:LspInfo
  • 重启LSP：:LspRestart
  • 启动LSP：:LspStart
  • 停止LSP：:LspStop
  • 打开Mason面板：:Mason
  • Mason安装：:MasonInstall <package> :MasonUninstall
  • Mason卸载：:MasonUninstall <package> :MasonUninstall
  • Mason更新：:MasonUpdate
  • Mason日志：:MasonLog

Lsp快捷键：

  • code action: <leader>ca
  • hover：K
  • 查看引用：gr
  • 查看定义：gd
  • 查看实现：gi
  • 查看声明：gD
  • 签名：<leader>ls
¶markdown
  • :MarkdownPreview
  • :MarkdownPreviewStop
