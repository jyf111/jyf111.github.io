现在有L,L+1,...R−1,RL, L+1,...R-1,RL,L+1,...R−1,R这R-L+1个数字，问你能够组成多少种不同的数字和
L≤109,R≤109L \leq 10^9, R \leq 10^9L≤109,R≤109

考虑用x个数字来组合
1个数字：[L,R][L, R][L,R]内的所有数字
2个数字：[L+L+1,R+R−1][L+L+1, R+R-1][L+L+1,R+R−1]内的所有数字
3个数字：[L+L+1+L+2,R+R−1−R−2][L+L+1+L+2, R+R-1-R-2][L+L+1+L+2,R+R−1−R−2]内的所有数字
则对于x个数字有：lx=x(2L+x−1)2,rx=x(2R−x+1)2l_x = \frac{x(2L+x-1)}{2}, r_x = \frac{x(2R-x+1)}{2}lx​=2x(2L+x−1)​,rx​=2x(2R−x+1)​
因此只要求用1~R-L+1个数字构成的各个区间的并集的长度即可
考虑lx>rx−1l_x>r_{x-1}lx​>rx−1​
则要满足x2+(L−R−2)x+R+1>0x^2 + (L-R-2)x + R + 1 > 0x2+(L−R−2)x+R+1>0
当(R−L+2)2−4R−4<0(R-L+2)^2 - 4R - 4<0(R−L+2)2−4R−4<0时，所有区间都不相交，又发现此时R-L+1在O(R\sqrt RR​)级别，所以暴力求出各个区间的长度相加即可
否则，求出两个根x1，x2，则在[x1,x2][x1,x2][x1,x2]这一段区间都是相交的，区间并的左端点为x1的左端点，右端点x2的右端点。
考虑R−L+2±(R−L+2)2−4R−42\frac{R-L+2 \pm \sqrt{(R-L+2)^2-4R-4}}{2}2R−L+2±(R−L+2)2−4R−4​​
设R-L+2为a，4R-4为b，且满足a≥ba \geq \sqrt ba≥b​
a−a2−b=a−(a+b)(a−b)≤a−(a−b)2=ba-\sqrt{a^2-b} = a-\sqrt{(a+\sqrt b)(a - \sqrt b)} \leq a - \sqrt{(a-\sqrt b)^2} = \sqrt ba−a2−b​=a−(a+b​)(a−b​)​≤a−(a−b​)2​=b​
所以1~x1的长度在O(b)O(\sqrt b)O(b​)级别，而x2~a的长度为a−R−L+2+(R−L+2)2−4R−42=R−L+2−(R−L+2)2−4R−42a-\frac{R-L+2 + \sqrt{(R-L+2)^2-4R-4}}{2} = \frac{R-L+2 - \sqrt{(R-L+2)^2-4R-4}}{2}a−2R−L+2+(R−L+2)2−4R−4​​=2R−L+2−(R−L+2)2−4R−4​​，也在O(b)O(\sqrt b)O(b​)级别，所以只需要对两端不相交的部分暴力即可
